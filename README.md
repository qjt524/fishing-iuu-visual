# 海洋非法捕捞(IUU)可视化分析系统
迷你挑战赛2 - 模式识别可视化解决方案

## 项目介绍
基于 HTML + CSS + JavaScript + ECharts + D3.js 实现渔业非法捕捞数据可视化，
通过热力图、折线图、柱状图、轨迹图等图表，识别涉嫌IUU非法捕捞的企业与船只。

## 使用技术
- 前端基础：HTML、CSS、JavaScript
- 可视化库：ECharts、D3.js

## 运行方式
1. 下载本项目中的 `fishing_visual.html` 文件
2. 使用任意现代浏览器（Chrome / Edge / 火狐）直接打开即可运行
3. 运行要求：电脑需联网（加载在线可视化库CDN）
